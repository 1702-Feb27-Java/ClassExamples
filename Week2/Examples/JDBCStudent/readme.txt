Simple Student example showcasing JDBC

Set Up:

1. run week_2_flash_card.sql on Database
2. import Student project into Eclipse
3. *IMPORTANT* Change username, password, 
and url in StudentDAOImpl & datasource.properties to match
your Database

Use:

1. Shows use of simple query & prepared statement.
2. Includes no main class, so could be used to 
teach students how to call a DAO implementation

Technologies: JRE 1.8 or newer | Eclipse Neon or newer | Oracle 11g | 
			  SQL Developer | JDBC 14
