jQuery Practice

ready()

jQuery provides the ready() function which will execute when the web page has fully loaded. Attach it to the document object: $(document).ready(function(){/*code to execute*/});

show(), hide(), toggle()

jQuery's show() and hide() functions show or hide an element within the DOM.

It is also possible to pass in an argument for the number of milliseconds over which the showing/hiding should take place over (animates the showing/hiding).

Alternatively, the toggle() function toggles the showing/hiding state of an element within the DOM. It also takes in a number-of-milliseconds argument.

append(), val()

jQuery's append() function adds an element as a child to another element.

jQuery's val() function gets the value of an input field. Passing in an argument, like val(12) assigns to it.

Input a value:

parents(), parentsUntil(); children(), find(); siblings(), next(), nextAll(), nextUntil(), prev(), prevAll(), prevUntil()

Parents of elements

jQuery's parents() function will return a list containing all the parent elements of an element. The parentsUntil() function takes in a string of a tag and will return all parents until it encounters that tag.

Children of elements

jQuery's children() function will return a list containing all the child elements of an element. The find() function takes in an selector argument for finding particular children that match the selector given.

Siblings of elements

jQuery's siblings() function will return a list containing all the sibling elements of an element.

readyState, readystatechange

The XMLHttpRequest object has 5 states, indexed 0 - 4, and the current state can be accessed in the readyState property. Each time this state changes, the readystatechange event is fired, so it is possible to act on a certain state. For example, you could write out the response only when the request and finished and a response returned.
